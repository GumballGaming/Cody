# cli-coding-agent

To install dependencies:

```bash
bun install
```

To run:

```bash
there are 2 ways 
1. do cd cli-coding-agent then bun run src
2. again do cd cli-coding-agent then run bun link then run cody then it can be used everywhere
```

This project was created using `bun init` in bun v1.3.6. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
